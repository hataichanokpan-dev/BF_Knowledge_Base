## 🎯 สูตรหลัก

```
┌─────────────────────────────────────────────────────────────┐
│                     ROIC CALCULATION                         │
├─────────────────────────────────────────────────────────────┤
│  ROIC = NOPAT / Invested Capital                            │
│                                                              │
│  NOPAT = EBIT × (1 - Tax Rate)                              │
│  Invested Capital = Equity + Debt - Cash                    │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                     WACC (THAILAND)                          │
├─────────────────────────────────────────────────────────────┤
│  WACC = (E/V × Ke) + (D/V × Kd × (1 - T))                   │
│                                                              │
│  Ke = Rf + β × ERP + CRP                                     │
│  Rf = 2.5-3% (Thai 10Y Bond)                                │
│  ERP = 6-7% | CRP = 2-3%                                    │
│  Total Ke = 10-13% typical                                   │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                      SPREAD SIGNAL                           │
├─────────────────────────────────────────────────────────────┤
│  ROIC - WACC > 5%  = ✅ STRONG BUY (if price right)         │
│  ROIC - WACC 1-5%  = ⚠️ HOLD / WATCH                        │
│  ROIC - WACC < 1%  = ❌ AVOID / SELL                         │
└─────────────────────────────────────────────────────────────┘
```

---

## 📊 ตัวเลขสำคัญ (Thai Market)

| Metric | ✅ Good | ⚠️ Warning | ❌ Danger |
|--------|---------|------------|-----------|
| **Spread (ROIC-WACC)** | > 5% | 1-5% | < 1% |
| **5Y Persistence** | 4+ yr | 2-3 yr | < 2 yr |
| **D/E Ratio** | < 1x | 1-2x | > 2x |
| **Interest Coverage** | > 8x | 4-8x | < 4x |
| **FCF Conversion** | > 80% | 50-80% | < 50% |
| **Net Debt/EBITDA** | < 2x | 2-3x | > 3x |

## 🏰 Moat Quick Check

```
Q1: ลูกค้าย้ายยาก? ──── Yes → SWITCHING COSTS
Q2: มีแบรนด์/IP? ────── Yes → INTANGIBLE ASSETS
Q3: ต้นทุนต่ำกว่า? ──── Yes → COST ADVANTAGE
Q4: ใช้ยิ่งมากยิ่งคุ้ม? ── Yes → NETWORK EFFECT
Q5: ตลาดเล็ก ผู้เล่นน้อย? Yes → EFFICIENT SCALE
```

**Moat Score (Quick):**
- ROIC-WACC Spread: ____/25
- Margin Stability: ____/20
- Persistence: ____/20
- FCF Conversion: ____/15
- Capital Discipline: ____/20
- **TOTAL: ____/100** (80+ = Wide, 60-79 = Narrow, <60 = None)

## 🚨 Value Trap Red Flags

**5 อันตรายที่สุด:**
1. ⚠️ ROIC < WACC 3+ ปีติด
2. ⚠️ D/E > 2.5x
3. ⚠️ FCF ติดลบ 2+ ปี
4. ⚠️ [[Related Party Transactions]] มาก
5. ⚠️ Goodwill impairment บ่อย

## 🔄 4-Step Screen

```
STEP 1: SPREAD ──► ROIC-WACC > 3% ──► 30-40 stocks
STEP 2: PERSIST ──► 3Y+ positive ──► 15-20 stocks
STEP 3: VALUE ───► P/E < Sector ──► 8-12 stocks
STEP 4: QUALITY ─► D/E < 3x ──────► 5-8 stocks
```

## 🛒 Buy/Hold/Sell Decision

| Condition | Action |
|-----------|--------|
| Spread > 5%, Wide Moat, P/FV < 0.8 | **BUY** (5-8%) |
| Spread > 5%, Narrow Moat, P/FV < 0.9 | **BUY** (3-5%) |
| Spread 1-5%, Moat OK, P/FV < 0.9 | **HOLD/WATCH** |
| Spread < 1% or No Moat | **AVOID/SELL** |
| P/FV > 1.2 | **TRIM 50%** |
| P/FV > 1.35 | **SELL ALL** |
