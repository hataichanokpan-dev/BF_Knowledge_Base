## โปรเจคนี้คืออะไร?

### สรุปสั้นๆ

> **"สร้างสูตรคัดเลือกหุ้นที่ให้ผลตอบแทนดีกว่าดัชนีตลาดหลักทรัพย์ไทย (SET) โดยมีความเสี่ยงต่ำกว่า"**

### อธิบายง่ายๆ แบบคนทั่วไปเข้าใจ

**คำถาม:** ถ้าคุณมีเงิน 1 ล้านบาท จะลงทุนยังไงให้ได้กำไรมากกว่าฝากธนาคาร แต่ไม่เสี่ยงเกินไป?

**คำตอบของโปรเจคนี้:**
- เราจะสร้าง "สูตร" ที่ช่วยคัดเลือกหุ้นดีๆ จากตลาดหลักทรัพย์ไทย
- สูตรนี้ใช้ **3 ปัจจัยหลัก** เพื่อคัดหุ้น:
  1. **ถูก (Value)** — หุ้นราคาถูกเมื่อเทียบกับคุณภาพ
  2. **ดี (Quality)** — บริษัทแกร่ง กำไรมั่นคง
  3. **มีแรง (Momentum)** — ราคาหุ้นกำลังขยับขึ้น

### โมเดล VQM คืออะไร?

| ตัวอักษร | ความหมาย | น้ำหนัก | คัดอะไร |
|:----------:|-----------|:--------:|----------|
| **V** | Value (ถูก) | 45% | หุ้นราคาถูกเมื่อเทียบกับกำไร/สินทรัพย์ |
| **Q** | Quality (ดี) | 35% | บริษัทมีกำไรมั่นคง หนี้น้อย |
| **M** | Momentum (แรง) | 20% | ราคากำลังขยับขึ้นใน 6 เดือนที่ผ่านมา |

## ทำไมต้องทำโปรเจคนี้

### ปัญหาที่พบในตลาดหลักทรัพย์ไทย

```
┌─────────────────────────────────────────────────────────┐
│  ปัญหา: นักลงทุนสถาบันในไทยส่วนใหญ่ทำผลงานได้แย่กว่า SET │
│                                                         │
│  - กองทุนรวม 80% ทำผลงานได้แย่กว่าดัชนี SET          │
│  - หลังหักค่าธรรมเนียม ผลตอบแทนต่ำลงอีก 2-4%           │
│  - การคัดหุ้นแบบเดิมๆ ไม่มีระบบที่ชัดเจน               │
└─────────────────────────────────────────────────────────┘
```

### ทำไมต้องใช้ 3 ปัจัยรวมกัน?

**ถ้าใช้ปัจจัยเดียว:**

| ปัจจัย | ข้อดี | ข้อเสีย |
|--------|--------|---------|
| **ซื้อแต่หุ้นถูก** | ได้หุ้นราคาถูก | อาจถูกตลอด ไม่มีวันขึ้น |
| **ซื้อแต่หุ้นดี** | ได้บริษัทแกร่ง | ราคาแพงเกินไป ผลตอบแทนต่ำ |
| **ซื้อแต่หุ้นมีแรง** | ได้หุ้นที่กำลังขึ้น | ซื้อสายแล้วอาจลงแรง |

**ถ้าใช้ 3 ปัจจัยรวมกัน (VQM):**
- ✅ ได้หุ้นถูก + ดี + มีแรง
- ✅ ลดความเสี่ยงจากการซื้อหุ้นที่โดนโกง
- ✅ เหมาะกับการลงทุนระยะยาว

                          │

                   └─────────────┘
```

### 3 ปัจจัยที่ใช้คัดหุ้น (รายละเอียด)

#### 1. Value (ถูก) — น้ำหนัก 45%

**คำถาม:** "หุ้นตัวนี้ราคาถูกหรือแพง?"

**เช็กจาก:**
- **P/E** — ราคาหุ้น ÷ กำไรต่อหุ้น (ยิ่งต่ำยิ่งถูก)
- **P/B** — ราคาหุ้น ÷ มูลค่าทางบัญชีต่อหุ้น
- **FCF Yield** — เงินสดที่บริษัทหาได้ ÷ มูลค่าบริษัท

**ตัวอย่าง:**
```
บริษัท A: P/E = 8 เท่า  → ถูก
บริษัท B: P/E = 25 เท่า → แพง
```

#### 2. Quality (ดี) — น้ำหนัก 35%

**คำถาม:** "บริษัทนี้แกร่งแค่ไหน?"

**เช็กจาก:**
- **ROIC** — อัตราผลตอบแทนต่อเงินลงทุน (ยิ่งสูงยิ่งดี)
- **หนี้ ÷ EBITDA** — หนี้เทียบกับกำไร (ยิ่งต่ำยิ่งดี)
- **Free Cash Flow** — เงินสดที่เหลือหลังจากลงทุน

**ตัวอย่าง:**
```
บริษัท A: ROIC = 18%, หนี้น้อย → ดี
บริษัท B: ROIC = 5%, หนี้เยอะ   → ไม่ดี
```

#### 3. Momentum (แรง) — น้ำหนัก 20%

**คำถาม:** "ราคาหุ้นกำลังไปทางไหน?"

**เช็กจาก:**
- **ผลตอบแทน 6 เดือน** — ราคาวันนี้เทียบกับ 6 เดือนก่อน
- **ปริมาณการซื้อขาย** — คนซื้อขายเยอะไหม

**ตัวอย่าง:**
```
บริษัท A: 6 เดือนที่ผ่านมา +15% → มีแรง
บริษัท B: 6 เดือนที่ผ่านมา -10% → ไม่มีแรง
```

### การรวมคะแนน (สมการง่ายๆ)

```
คะแนน VQM = (0.45 × คะแนนถูก) + (0.35 × คะแนนดี) + (0.20 × คะแนนแรง)

เช่น:
หุ้น ABC: ถูก(80) + ดี(70) + แรง(60)
คะแนนรวม = (0.45 × 80) + (0.35 × 70) + (0.20 × 60)
           = 36 + 24.5 + 12
           = 72.5 คะแนน
```

**ยิ่งคะแนนสูง ยิ่งดี → คัดเลือกเข้าพอร์ต**

## เป้าหมายของโปรเจค

### เป้าหมายหลัก (ต้องทำได้)

| เป้าหมาย | ตัวเลข | แปลว่าอะไร |
|-----------|---------|--------------|
| **ผลตอบแทนเกิน SET** | > 3% ต่อปี | ทำกำไรได้มากกว่าลงทุนตามดัชนี |
| **ความเสี่ยงต่ำกว่า** | Max DD < -25% | ตกแรงสุดไม่เกิน 25% (SET ตก -35% ในปี 2020) |
| **Sharpe Ratio** | > 1.0 | ผลตอบแทนดีเมื่อเทียบกับความเสี่ยง |
| **น่าเชื่อถือทางสถิติ** | p < 0.05 | ไม่ใช่ดวง แต่เป็นสูตรที่ใช้ได้จริง |

### ขอบเขตของการวิจัย

| รายการ | รายละเอียด |
|---------|-------------|
| **ตลาด** | ตลาดหลักทรัพย์ไทย (SET) เฉพาะหลัก |
| **ช่วงเวลา** | ปี 2019 - 2024 (5 ปี) |
| **จำนวนหุ้น** | 20-30 ตัวต่อพอร์ต |
| **ปรับพอร์ต** | ทุก 3 เดือน |
| **ข้อจำกัด** | 1 หุ้นไม่เกิน 5%, 1 กลุ่มอุตสาหกรรมไม่เกิน 30% |

### รายละเอียดแต่ละ Phase

#### Phase 1: เตรียมข้อมูล (เดือนที่ 1-2)

```
✅ ติดตั้งโปรแกรม Python
✅ ดาวน์โหลดข้อมูลหุ้นย้อนหลัง 5 ปี
✅ กำหนดสูตรคำนวณทั้งหมด
✅ วางกฎการคัดหุ้น
```

**ข้อมูลที่ต้องการ:**
- ราคาหุ้นปิดทุกวัน
- ราคาปริมาณการซื้อขาย
- งบการเงิน (รายได้ กำไร หนี้)
- ข้อมูลเกี่ยวกับปันผล แบ่งปันผล

#### Phase 2: สร้างสูตรคัดเลือก (เดือนที่ 3-8)

```
เขียนโค้ดคำนวณ:
├── Value Factors (สูตรถูก)
├── Quality Factors (สูตรดี)
├── Momentum Factors (สูตรแรง)
└── รวมคะแนน VQM Score
```

**Output:** ได้ระบบที่ใส่วันที่ → ได้รายชื่อหุ้นที่ควรซื้อ

#### Phase 3: ทดสอบย้อนหลัง (เดือนที่ 9-13)

```
ทดสอบกับข้อมูลจริงย้อนหลัง 5 ปี:

ปี 2019 → คัดหุ้น → ดูผล 3 เดือน → บันทึก
ปี 2020 → คัดหุ้น → ดูผล 3 เดือน → บันทึก
...
ปี 2024 → คัดหุ้น → ดูผล 3 เดือน → บันทึก

เปรียบเทียบ:
├── VQM Portfolio vs SET Index
├── VQM vs ซื้อแต่หุ้นถูก
├── VQM vs ซื้อแต่หุ้นดี
└── VQM vs ซื้อแต่หุ้นมีแรง
```

**Output:** ตารางผลลัพธ์ + กราฟ + สถิติ

#### Phase 4: เขียนวิจัย (เดือนที่ 14-20)

```
บทที่ 1: แนะนำ (Introduction)
├── บอกว่าทำไมถึงทำ
├── บอกว่าจะทำอะไร
└── บอกว่าคาดหวังผลอะไร

บทที่ 2: ทบทวนวรรณกรรม (Literature Review)
├── หาข้อมูลเก่าๆ ที่คนคนอื่นทำมาแล้ว
├── สรุปว่าใครทำอะไรไปบ้าง
└── บอกว่าเราต่างจากคนอื่นยังไง

บทที่ 3: วิธีการ (Methodology)
├── บอกทีละขั้นตอนว่าทำยังไง
├── ใส่สูตรทั้งหมด
└── ให้คนอื่นทำตามได้

บทที่ 4: ผลลัพธ์ (Results)
├── แสดงกราฟทั้งหมด
├── แสดงตารางทั้งหมด
└── อธิบายว่าเจออะไร

บทที่ 5: สรุป (Conclusion)
├── ตอบคำถามวิจัย
├── สรุปสิ่งที่ค้นพบ
└── ข้อเสนอแนะ
```

#### Phase 5: แก้ไข & ตรวจสอบ (เดือนที่ 21-26)

```
✅ ส่งให้ที่ปรึกษาตรวจ

## คำถามที่พบบ่อย (FAQ)

### Q1: ถ้าทำได้จริง ทำไมทุกคนไม่ทำกันหมด?

**A:** หลายคนทำอยู่แล้ว! แต่:
- ต่างประเทศมีมาก (แต่ไทยยังไม่ค่อยมี)
- สูตรมีหลายแบบ บางอันซับซ้อนมาก
- เราจะทำแบบง่าย โปร่งใส เหมาะกับไทย

### Q2: ต้องมีพื้นฐานการเงินแค่ไหน?

**A:** ถ้าอ่านเอกสารนี้แล้วเข้าใจ พอแล้ว!
- เข้าใจ P/E, P/B พอสมควร
- เข้าใจว่าหุ้นคืออะไร
- รู้จักตลาดหลักทรัพย์ไทย

### Q3: ใช้เงินจริงทดสอบไหม?

**A:** ไม่! เราใช้ข้อมูลจริงย้อนหลัง 5 ปี มาจำลอง (Backtest)
- เหมือนดูหนังย้อนหลัง แล้วทำนายว่าถ้าทำตอนนั้นจะเกิดอะไร
- ไม่เสียเงินจริง

### Q4: รับประกันว่าจะได้กำไรหรือไม่?

**A:** ไม่รับประกัน!
- นี่คือการวิจัย ไม่ใช่การแนะนำลงทุน
- อดีตไม่ได้บอกอนาคตเสมอไป
- แต่จะพยายามหาสูตรที่ใช้ได้จริง
