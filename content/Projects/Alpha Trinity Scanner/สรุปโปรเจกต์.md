# Alpha Trinity Scanner - สรุปโปรเจกต์

> *"Better roughly right than precisely wrong with broken data"* - Damodaran

## ภาพรวม

**Alpha Trinity Scanner** คือระบบคัดกรองหุ้นไทยโดยใช้ **Reverse DCF** เพื่อวัด "Expectation Gap" (ช่องว่างระหว่างความคาดหวังของตลาดกับความสามารถจริง)

### สมมติฐานหลัก

1. **ราคาหุ้นสะท้อนความคาดหวัง** (implied growth, margin, ROIC)
2. **Growth Traps**: ความคาดหวังสูงเกินไป → ผลงานต่ำกว่า
3. **Value Opportunities**: ความคาดหวังต่ำเกินไป → ผลงานเหนือกว่า

## สัญญาณ (Signals) - ถูกล็อกแล้ว

```
AVOID:      composite > 50%    (ความคาดหวังสูงเกินไป)
CAUTION:    20% < comp ≤ 50%   (ความคาดหวังปานกลาง)
ACCEPTABLE: composite ≤ 20%    (ความคาดหวังพอเหมาะ)
```

### Composite Score Formula

```
composite = (0.40 × growth_gap) +
           (0.30 × margin_gap) +
           (0.30 × roic_gap)
```

## สิ่งที่ค้นพบสำคัญ (Key Findings)

### 1. CAUTION Zone = Sweet Spot ⭐

**การค้นพบ**: หุ้นกลุ่ม CAUTION (20-50% gap) ให้ผลตอบแทนสูงสุด

| Signal | ผลตอบแทนเฉลี่ย |
|--------|-------------------|
| CAUTION | **+16.59%** 🏆 |
| ACCEPTABLE | +13.63% |
| AVOID | -2.35% |

**คำอธิบาย**:
- สถาบันการลงทุนไทยชอบ "quality at reasonable price"
- ถูกเกินไป = value trap (ถูกละเลย)
- แพงเกินไป = priced for perfection (เสี่ยงผิดหวัง)
- พอดี = institutional favorites

### 2. Data Triangulation = Competitive Moat

**ปัญหา**: yfinance คืนค่า EBIT = 0 สำหรับหุ้นไทยทุกตัว

**วิธีแก้**: ดึงข้อมูลโดยตรงจากรายงานการเงิน
```python
# ก่อน: yfinance ผิด
>>> yfinance.Ticker("PTTGC.BK").info.get('ebitda')
0  # ผิด!

# หลัง: ดึงจาก statements
>>> dt.extract_financials("PTTGC")
{'ebit': 45234000000, 'operating_margin': 0.0661}  # ถูกต้อง!
```

### 3. Signal Mechanism ได้รับการยืนยัน

**สหสัมพันธ์ -30.35%**:
- Gap สูง → ผลตอบแทนต่ำ
- Gap ต่ำ → ผลตอบแทนสูง
