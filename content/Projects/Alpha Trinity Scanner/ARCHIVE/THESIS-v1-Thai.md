## 1. บทนำ

### 1.1 บริบทและแรงจูงใจการวิจัย

**ตลาดหลักทรัพย์แห่งประเทศไทย (SET)** นำเสนอสภาพแวดล้อมที่เป็นเอกลักษณ์สำหรับการวิจัยการประเมินมูลค่าหุ้น ในฐานะตลาดเกิดใหม่ที่มีมูลค่าตลาดประมาณ **500 พันล้านเหรียญสหรัฐ** (ปี 2025) ประเทศไทยมีลักษณะเฉพาะที่แตกต่างจากตลาดพัฒนาแล้ว:

```
┌─────────────────────────────────────────────────────────────┐
│           ลักษณะเฉพาะของตลาดหลักทรัพย์ไทย                   │
├─────────────────────────────────────────────────────────────┤
│  มูลค่าตลาด (2025):        ~500 พันล้านเหรียญสหรัฐ    │
│  ช่วงดัชนี SET (2025-26):    1,400-1,500                  │
│  อัตราไร้ความเสี่ยง:        2.5% (พันธบัรรัฐบาล 10 ปี)   │
│ ตัวชี้ความเสี่ยงตลาด:      6.0% (ตลาดเกิดใหม่)        │

│  การครอบคลุมนักวิเคราะห์:   ต่ำกว่าตลาดพัฒนาแล้ว      │
```

## 2. บทความวรรณกรรม

### 2.1 รากฐานทฤษฎี

#### 2.1.1 ทฤษฎีการประเมินมูลค่าสมัยใหม่

**สมการหลัก:**

$$P_0 = \sum_{t=1}^{n} \frac{FCF_t}{(1+WACC)^t} + \frac{TV_n}{(1+WACC)^n}$$

โดยที่:
- $P_0$ = ราคาตลาดปัจจุบัน
- $FCF_t$ = กระแสเงินสดอิสระในช่วงเวลา $t$
- $WACC$ = ต้นทุนทุนถ่วงเฉลี่ยถ่วงน้ำหนัก
- $TV_n$ = มูลค่าถาวรในช่วงเวลา $n$

#### 2.1.2 สเปกตรัมประสิทธิภาพข้อมูล

```
┌─────────────────────────────────────────────────────────────┐
│           การจำแนกรูปแบบประสิทธิภาพตลาด               │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  รูปแบบอ่อน      → ราคาอดีตไม่สามารถทำนายอนาคตได้  │
│  รูปแบบกึ่งแข็ง   → ข้อมูลสาธารณะสะท้อนทันที          │
│  รูปแบบแข็ง      → ข้อมูลทั้งหมด (แม้กระทั้งส่วนตัว)  │
│                      สะท้อนแล้ว                              │
│  การจำแนนไทย:    รูปแบบกึ่งแข็ง (พร้อมช่องว่าง    │
│                    ประสิทธิภาพบางส่วน)                      │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 วิธีการ Reverse DCF

**งานบุกเบิก:**

| นักวิจัย | การมีส่วนร่วม | ข้อเข้าใจสำคัญ |
|----------|-----------------|------------------|
| **ดาโมดารัน (2006)** | เฟรมเวิร์ก DCF ผกผัน | ราคา → ความคาดหวังที่ฝัง |
| **เมาบูสซิน (2006)** | การลงทุนแบบคาดหวัง | ขนาดช่องว่างทำนายผลตอบแทน |
| **เวตเตอร์ (2015)** | การระบุ growth trap | P/E สูง + ความคาดหวังสูง = ประสิทธิภาพต่ำ |

### 2.3 ทฤษฎีช่องว่างความคาดหวัง

**ข้อเสนอหลัก:**

$$R_t = \alpha - \beta \times Gap_{t-1} + \epsilon_t$$

โดยที่:
- $R_t$ = ผลตอบแทนในภายหลัง
- $Gap_{t-1}$ = ช่องว่างความคาดหวังในช่วงก่อนหน้า
- $\beta > 0$ (สัมประสิทธิ์เชิงบวก)

### 3.3 Reverse DCF Engine

**การจัดรูปคณิตศาสตร์:**

กำหนด:
$$EV_{market} = \sum_{t=1}^{T} \frac{FCF_0(1+g)^t}{(1+WACC)^t} + \frac{FCF_0(1+g)^T(1+g_T)}{(WACC-g_T)(1+WACC)^T}$$

แก้หา $g$ โดยใช้ Newton-Raphson:

$$g_{n+1} = g_n - \frac{f(g_n)}{f'(g_n)}$$

โดยที่:
$$f(g) = DCF(g) - EV_{market}$$

### 3.4 เครื่องคิดเลขความสามารถสมจริง

**แนวทางการเปรียบเทียบมาตรฐานอุตสาหกรรม:**

$$Cap_{สมจริง} = Cap_{อุตสาหกรรม} \times (1 + \alpha_{สถานะ})$$

**ปัจจัยการปรับสถานะ ($\alpha_{สถานะ}$):**

| **ขยายตัว** | +0.20 | 0.00 | -0.10% |
| **ปกติ** | 0.00 | 0.00 | 0.00% |
