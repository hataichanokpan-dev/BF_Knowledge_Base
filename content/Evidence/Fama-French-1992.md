## 🔬 Core Finding

### The HML Factor (High Minus Low)

Fama & French ระบุว่า **Book-to-Market ratio** เป็นตัวทำนายผลตอบแทนหุ้นได้ดีกว่า Beta เพียงอย่างเดียว

```
3-Factor Model:
Rp - Rf = α + β(Mkt-Rf) + s*SMB + h*HML + ε

Where:
- SMB = Small Minus Big (size factor)
- HML = High Minus Low (value factor)
```

### Key Results

| Finding | Data |
|---------|------|
| Value Premium | ~4-6% annually (high B/M vs low B/M) |
| Persistence | 1940s - 2000s, multi-decade evidence |
| Risk Explanation | Value = compensation for distress risk |
| Statistical Significance | t-stat > 3.0 for HML |


### Risk-Adjusted Performance

| Metric | Value Portfolio | Growth Portfolio |
|--------|-----------------|------------------|
| **Raw Return** | 17.2% | 8.4% |
| **CAPM Alpha** | +3.8% | -2.1% |
| **3-Factor Alpha** | +0.4% | -0.3% |
| **Beta** | 1.15 | 1.10 |

**Key Insight:** CAPM alpha หายไปใน 3-factor model = Value premium ถูกอธิบายโดย HML factor

## 💡 Interpretation

### Fama-French View: Risk-Based

> "Value premium เป็นค่าตอบแทนจากความเสี่ยง (distress risk) ที่สูงขึ้น"

**Logic:**
1. หุ้น Value มักมี B/M สูงเพราะราคาตก (distress signal)
2. นักลงทุนต้องการค่าตอบแทนสูงขึ้นสำหรับหุ้นที่มีความเสี่ยง
3. จึงเป็น rational pricing ไม่ใช่ market inefficiency

**Implication:** คุณรับความเสี่ยงเพิ่ม = คุณควรได้รับค่าตอบแทนเพิ่ม

## ⚠️ Criticisms

| Criticism | Response |
|-----------|----------|
| **Data Mining** | Out-of-sample tests in international markets confirm |
| **Survivorship Bias** | Results hold with delisted stocks included |
| **Transaction Costs** | Premium remains positive after costs |
| **Behavioral Alternative** | Lakonishok et al. (1994) argue it's mispricing |
