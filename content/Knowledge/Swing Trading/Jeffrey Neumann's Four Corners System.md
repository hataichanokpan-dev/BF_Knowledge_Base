## แนะนำตัว — ผมคือ Jeffrey Neumann

ผมเปลี่ยน $2,500 เป็น $50M+ ในชีวิตการเทรดของผม นั่นคือ 2,000,000% return แต่สิ่งที่สำคัญกว่าคือ **ผมไม่เคยโดน Cut จากเกม** ไม่เหมือนคนส่วนใหญ่ที่รวยแล้วโดนฟาดหมด

**สิ่งสำคัญที่สุด:** ผมไม่ได้ "เล่นตลาด" ผม **รอ** ให้ทุกอย่างเรียงตัวสวยก่อน คนส่วนใหญ่รีบเข้า ผมรอ คนส่วนใหญ่ FOMO ผมค่อยเข้า

> "ผมไม่ได้เล่นตลาด ผมรอให้ทุกอย่างเรียงตัวสวย"

## Part 1 — Four Corners Framework: 4 เกณฑ์ที่ต้องพร้อมพร้อมกัน

ผมไม่มองหุ้นเป็นตัวๆ ไป แต่มองว่าหุ้นแต่ละตัวต้องผ่าน "4 มุม" นี้พร้อมกัน ขาดมุมไหน = ไม่เข้า (รอต่อ)

### เปรียบเทียบ: Four Corners เหมือนโต๊ะอาหาร 4 ขา

```
ถ้าโต๊ะมี 4 ขา → นั่งได้สบาย
ถ้าโต๊ะขาด 1 ขา → โต๊ะล้ม
ถ้าหุ้นขาด 1 Corner → เทรดล้ม
```

### Four Corners Matrix

```
┌─────────────┬─────────────┐
│   CHART     │  STRUCTURE  │
│ (Technical) │ (Low Float) │
├─────────────┼─────────────┤
│   SECTOR    │  CATALYST   │
│   (Theme)   │   (Story)   │
└─────────────┴─────────────┘
```

| Corner | คำถามหลัก | หาย ไม่ผ่าน |
|--------|-----------|-------------|
| **CHART** | "หุ้นตัวนี้กำลังตื่นจากการพักตัวไหม?" | ยังไม่มี Breakout |
| **STRUCTURE** | "โครงสร้างหุ้นตัวนี้พร้อมระเบิดไหม?" | Float ใหญ่/Dilution |
| **SECTOR** | "เงินกำลังไหลเข้า sector นี้ไหม?" | ตัวเดียวพุ่ง |
| **CATALYST** | "ทำไมราคาต้องเคลื่อนไหวตอนนี้?" | ไม่มีเหตุผลชัดเจน |

### เปรียบเทียบ: Neumann vs คนทั่วไป

| คนทั่วไป | Neumann (ผม) |
|-----------|--------------|
| เห็นหุ้นตัวหนึ่งดี → ซื้อเลย | เห็นหุ้นดี → รอ 4 Corners align |
| ไม่สนใจ Sector | ต้องมี Sector Momentum |
| ไม่อ่าน Dilution | อ่านทุก Filing |
| ใช้ Hard Stop | ใช้ Mental Stop ตาม Thesis |
| เล่นทุกวัน | รอจังหวะที่ดีที่สุด |

## Part 2 — WAVE 1: STRUCTURAL FILTER (โครงสร้างพร้อมระเบิด)

คนส่วนใหญ่มองถึงราคาและ Chart ก่อน ผมมอง **โครงสร้าง** ก่อน เพราะโครงสร้างแย่ = ไม่ว่า Chart ดีแค่ไหนก็ระเบิดไม่ได้

### เปรียบเทียบ: โครงสร้างหุ้น เหมือน รถยนต์

```
โครงสร้างดี   = Ferrari สภาพดี → เตาะก็บิน
โครงสร้างแย่ = Ferrari พัง → กดแก๊สก็ไม่ไป
```

### 5 เกณฑ์โครงสร้าง (Daily Batch)

**① Float ต้องน้อย**

ทำไม Float ต่ำสำคัญ? เพราะ Float ต่ำ = คนถือน้อย เมื่อคนมาซื้อพร้อมกัน ราคาบินได้แรงมาก

```
เปรียบเทียบ:
Float 100M = คน 100 คนแบ่งกินเค้ก 1 ผืน → คนละนิดเดียว
Float 5M   = คน 5 คนแบ่งกินเค้ก 1 ผืน → คนละเยอะมาก
```

| Float | Rating | ความหมาย |
|-------|--------|----------|
| < 10M | ★★★ | ระเบิดได้ทุกเมื่อ แต่ bid-ask กว้าง |
| 10-50M | ★★★ | ดีเยี่ยม ศักยภาพสูง |
| 50-200M | ★★ | ผ่านขั้นต่ำ |
| > 200M | ★ | ตัดออก (ต้องการเงินเยอะเกินไป) |

**ตรวจสอบได้ที่:** Finviz → Shs Float

**② Insider Ownership (Skin in the Game)**

ทำไม Insider สำคัญ? เพราะถ้าผู้บริหารไม่ถือหุ้นตัวเอง = เขาไม่สนใจว่าราคาจะไปไหน

```
Insider 30%+ = ผู้บริหารมีเลือดเหลืองกับหุ้นนี้
Insider < 15% = ผู้บริหารถือเหมือนคนธรรมดา = ไม่แคร์
```

| Insider % | Rating | ความหมาย |
|-----------|--------|----------|
| > 30% | ★★★ | ดีมาก ผู้บริหารมีแรงจูงใจสูง |
| 15-30% | ★★ | ผ่านขั้นต่ำ |
| < 15% | ★ | ไม่มี skin in the game |

**③ Institutional Ownership ต่ำ**

ทำไหม Institutional ต่ำดี? เพราะ Institutional ต่ำ = คุณเข้าก่อน Big Money

```
Institutional < 10% = ยังไม่มีสถาบันเข้ามา = โอกาสร่วมกับ Smart Money
Institutional > 30% = สถาบันถือเยอะแล้ว = รอช้าไปแล้ว
```

| Institutional % | Rating | ความหมาย |
|-----------------|--------|----------|
| < 10% | ★★★ | แทบไม่มีสถาบันถือเลย |
| 10-20% | ★★ | ดีมาก |
| 20-30% | ★ | ผ่านขั้นต่ำ |
| > 30% | ✗ | สถาบันถือเยอะเกินไป |

**④ Price Range ($0.50 - $10)**

| Price Range | ปัญหา |
|-------------|-------|
| < $0.50 | มักเป็นหุ้นที่ "กำลังจะตาย" อาจถูก delist |
| $0.50 - $10 | ✅ Sweet spot สำหรับ micro-caps |
| > $10 | ต้องการเงินทุนมากขึ้น ศักยภาพจำกัด |

**⚠️ ความผิดพลาด:** ผมเคยซื้อหุ้น $0.30 ที่ดูถูก แต่มันไปไหนไม่ได้ เพราะคนกลัวว่าจะตาย (Going Concern)

**⑤ ไม่มี Dilution ล่าสุด**

**Dilution = ศัตรูร้ายแรงที่สุด** บริษัทขายหุ้นเพิ่ม = ราคาโดนกด

| Filing | ความหมาย | Impact |
|--------|----------|--------|
| S-3 (Shelf) | อาจขายได้ทุกเมื่อ | -30 pts |
| 424B (Prospectus) | กำลังขายอยู่ | -50 pts |
| ATM Offering | ขายทีละน้อยทุกวัน | -50 pts |

**Keywords อันตรายใน Filing:**
```
- "At-The-Market"
- "ATM offering"
- "From time to time we may offer"
- "Sales Agreement with [Broker]"
- "Equity Line of Credit"
```

**ตรวจสอบได้ที่:** SEC EDGAR → ค้นหา Ticker → ดู S-3, 424B, 8-K

## Part 3 — WAVE 2: TECHNICAL IGNITION (จับจังหวะตื่น)

โครงสร้างดีแล้ว ตอนนี้รอจังหวะ "ตื่น" ผมเรียกมันว่า **Technical Ignition**

### เปรียบเทียบ: Technical Ignition เหมือน จุดระเบิด

```
โครงสร้างดี = ดินระเบิดวางไว้แล้ว
Technical Ignition = จุดระเบิดติดไฟ
```

### 5 เกณฑ์ทางเทคนิค (Realtime)

**① Long-Term Basing Pattern**

หุ้นที่จะบินต้อง "นอนตัว" นานๆ ก่อน เพื่อให้คนที่ไม่อยากถือออกไปหมด

```
ราคา
  │
$5│     ╱╲
  │    ╱  ╲
$3│   ╱    ╲────────────────────╱╲
  │  ╱                      ╱╲ ╱  ╲
$1│ ╱      Basing Zone     ╱  ╲    ╲──► Breakout!
  │╱   (1-2 years of death)╱
  └────────────────────────────────────────► เวลา
```

| Basing Period | Rating |
|---------------|--------|
| > 365 days | ★★★ ดีมาก |
| 180-365 days | ★★ ผ่าน |
| < 180 days | ★ ยังไม่พร้อม |

**② Price Compression (Squeeze)**

Squeeze คืออะไร? คือราคาถูกบีบอัดในกรอบแคบๆ เหมือนสปริง

```
Bollinger Band บอก Squeeze ได้:
BB Width = (Upper Band - Lower Band) / Middle Band
```

| BB Width Percentile | Rating |
|--------------------|--------|
| < 10th | ★★★ Super Squeeze |
| 10-20th | ★★ Squeeze |
| > 20th | ★ ยังไม่บีบอัดพอ |

**③ RVOL Spike (Relative Volume)**

Volume = Footprint ของ Smart Money ถ้าไม่มี Volume สูง Breakout ปลอม

```
RVOL = Volume วันนี้ / Volume เฉลี่ย 20 วัน

ตัวอย่าง:
- Volume วันนี้: 50 ล้านหุ้น
- Volume เฉลี่ย 20 วัน: 5 ล้านหุ้น
- RVOL = 50 / 5 = 10x
```

| RVOL | Rating | Action |
|------|--------|--------|
| > 10x | ★★★ | สัญญาณแรงมาก |
| 5-10x | ★★ | สัญญาณแรง |
| 2-5x | ★ | เริ่มสนใจ |
| < 2x | ✗ | ปกติ |

**④ RSI Historic Oversold**

RSI Oversold = หุ้นถูกทุ่มจนเกินไป = โอกาส Bounce

| RSI Level | Rating |
|-----------|--------|
| < 20 (historic) | ★★★ Extreme oversold |
| 20-30 (first time in year) | ★★ Historic oversold |
| 30-40 | ★ Oversold ปกติ |
| > 40 | ✗ ไม่ oversold |

**⑤ Downtrend Line Breakout + Volume Confirm**

Breakout ต้องมากับ Volume สูง ไม่งั้นปลอม

| Breakout แบบไหน | Volume | ผลลัพธ์ |
|----------------|--------|---------|
| Breakout + Volume สูง | RVOL > 5x | ✅ Valid breakout |
| Breakout + Volume ต่ำ | RVOL < 1x | ✗ Fake-out (หลอก) |

## Part 4 — WAVE 3: SECTOR CLUSTER (เงินไหลเข้ากลุ่ม)

นี่คือ **Secret Sauce** ที่แยกผมจากคนอื่น ผมไม่ซื้อหุ้นเดี่ยว ผมซื้อ **Sector**

### เปรียบเทียบ: Sector Cluster เหมือน คลื่นน้ำ

```
หุ้นเดี่ยวพุ่ง = หยดน้ำตก → อาจเป็นความบังเอิญ
Sector พุ่ง   = คลื่นน้ำทะเล → แน่นอนว่ามีแรง
```

### ตัวอย่าง Sector Rotation ที่ผมเคยเล่น

**① 3D Printing (2012-2013)**

```
Timeline:
Jan 2012: DDD, SSYS เริ่ม Base Breakout
Apr 2012: XONE, ONVO ตาม Breakout → 3+ stocks = Cluster confirmed
Jul 2012: VJET IPO → Sector Mania begins
Jan 2013: Golf Course people talking about 3D printing = EXIT
```

**Lesson:** เข้าตอน Stealth (Jan 2012), ออกตอน Mania (Jan 2013)

**② Cannabis (2018-2019)**

```
Timeline:
Jan 2018: Canada legalization bill introduced
Apr 2018: TLRY, CGC, CRON Breakout together
Jul 2018: Sector euphoria, every stock up 200%+
Oct 2018: Canada legalized = Sell on news
Dec 2018: Sector crashed 70%+
```

**Lesson:** Buy Rumor (Jan 2018), Sell News (Oct 2018)

**③ EV/Bitcoin (2020-2021)**

```
Timeline:
Mar 2020: COVID crash, EV stocks dead
Jun 2020: TSLA Breakout + SPLIT announcement
Sep 2020: NKLA, WKHS, BLNK follow
Jan 2021: Retail FOMO every EV stock
Feb 2021: Mania top
```

**Lesson:** Lead Dog (TSLA) บอกทิศทาง Sector ได้เสมอ

### 4 เกณฑ์ Sector (Realtime)

**① Cluster Breakout (3+ stocks)**

| จำนวนหุ้นพุ่ง | ความหมาย |
|--------------|----------|
| 1 ตัว | อาจเป็นข่าวเฉพาะตัว |
| 2 ตัว | อาจเป็นความบังเอิญ |
| 3+ ตัว | ✅ เงินไหลเข้า sector แน่นอน! |

**⚠️ ความผิดพลาด:** ผมเคยซื้อหุ้นเดี่ยวที่ดูเจ๋ง แต่ Sector ไม่ไปไหน สุดท้ายราคากลับมาเท่าเดิม

**② Relative Strength (หา Lead Dog)**

Lead Dog = หุ้นที่แรงที่สุดใน Sector = ซื้อตัวนี้หนักสุด

```
RS = (ราคาหุ้น / ราคา Index) / เฉลี่ย 50 วัน

ตัวอย่าง:
Sector: 3D Printing
├── DDD:  RS = 2.1  ◄── Lead Dog!
├── SSYS: RS = 1.8
├── XONE: RS = 1.5
├── ONVO: RS = 1.2
└── VJET: RS = 0.9

ซื้อทุกตัว → Pyramid เข้า DDD (Lead Dog)
```

**③ Sector Lifecycle Phase**

รู้ว่า Sector อยู่ Phase ไหน = รู้ว่าควรซื้อหรือขาย

```
Phase 1: STEALTH
├── Smart money เข้า
├── ยังไม่มีใครพูดถึง
├── Volume ต่ำ
├── Media เงียบ
└── ★ ENTER HERE ★

Phase 2: AWARENESS
├── Early adopters เข้า
├── Media เริ่มพูดถึง
├── Volume เริ่มขึ้น
├── Analysts เริ่ม Cover
└── ★ ADD HERE ★

Phase 3: MANIA
├── Mass public FOMO
├── ทุกคนพูดถึง (Taxi driver, Golf course)
├── Volume พุ่ง
├── Taxi drivers คุยเรื่องนี้
└── ★ EXIT HERE ★

Phase 4: BLOW-OFF
├── Crash
├── Volume แห้ง
├── Everyone hates it
└── ★ AVOID ★
```

> 🔑 **"Golf Course Indicator":** ถ้าได้ยินคนทั่วไปพูดถึง sector นั้นบนสนามกอล์ฟ = เข้าสู่ Mania Phase = EXIT

**④ Shotgun Approach**

วิธีซื้อ Sector ของผม:

```
1. ระบุ Hot Sector
        ↓
2. ซื้อ Basket 5-10 ตัวใน sector
        ↓
3. ดูว่าตัวไหนแรงสุด = Lead Dog
        ↓
4. Pyramid เข้า Lead Dog
        ↓
5. ตัดตัวที่อ่อน (Cut laggards)
```

## Part 5 — WAVE 4: CATALYST AUDIT (เหตุผลเคลื่อนไหว)

หุ้นดีต้องมี "เรื่องเล่า" ที่ทำให้ราคาเคลื่อนไหว ผมเรียกมันว่า **Catalyst**

### เปรียบเทียบ: Catalyst เหมือน น้ำมันเชื้อเพลิง

```
โครงสร้างดี + Technical ดี = รถพร้อมแล้ว
Catalyst = น้ำมัน = รถวิ่ง!
```

| Legislative | Bill vote date | Congress calendar | Exact date known |
| Earnings | Quarterly report | Earnings calendar | Exact date known |
| Product Launch | iPhone release | Company IR | Approximate date |
| Court Ruling | Patent case | Court docket | Exact date known |

**② Volume Catalyst (แรงรองลงมา)**

```
Volume พุ่ง 200 ล้านหุ้น
        ↓
ไม่มีข่าวอะไร
        ↓
แสดงว่า: มีคนรู้อะไรบางอย่าง
        ↓
Follow the smart money!
```

**③ Thematic Catalyst**

|-------|----------|----------|
| AI | ChatGPT launch | Nov 2022 - Ongoing |
| Clean Energy | Government subsidies | 2021-2022 |

| Cannabis | Legalization | 2018-2019 |

**④ Buy the Rumor, Sell the News**

กฎทองคำของผม:

```
Timeline:
─────────────────────────────────────────────────────
         │                    │
    Buy Zone              Event          Sell Zone
    (Rumor)                              (News)
         │                    │              │
    ██████████████████████   │   ▓▓▓▓▓▓▓▓▓▓▓▓
    (Accumulation)           │   (Distribution)
                             │
                        Event จริง
```

**กฎ:**
- **ก่อน Event** = BUY (Rumor phase) — คนเริ่มรู้ แต่ยังไม่ Mass adoption
- **วัน Event** = SELL (News phase) — ข่าวออก = คนขายข่าว
- **หลัง Event** = AVOID (Priced in) — ราคา Already reflected news

> 💡 **ความผิดพลาด:** ผมเคยถือหุ้นหลัง FDA Approved คิดว่าจะวิ่งต่อ สุดท้ายราคา Dump เพราะคนขายข่าว

## Part 6 — NEUMANN SCORE: รวมทุก Wave

เมื่อ 4 Corners ผ่านหมด ผมจะให้คะแนน เพื่อตัดสินใจว่าคุ้มไหม

### Formula

```
Neumann Score = (W1 × 0.20) + (W2 × 0.30) + (W3 × 0.20) + (W4 × 0.30)

W1 = Structure Score (Wave 1)
W2 = Technical Score (Wave 2)
W3 = Sector Score (Wave 3)
W4 = Catalyst Score (Wave 4)
```

### ทำไมน้ำหนักไม่เท่ากัน?

| Wave | Weight | เหตุผล |
|------|--------|--------|
| W2 (Technical) | 30% | จับจังหวะเข้าสำคัญที่สุด |
| W4 (Catalyst) | 30% | เหตุผลเคลื่อนไหวสำคัญที่สุด |
| W1 (Structure) | 20% | โครงสร้างดี แต่ไม่จำเป็นต้องสมบูรณ์ |
| W3 (Sector) | 20% | ช่วยได้ แต่หุ้นเดี่ยวก็ได้ |

### Neumann Score Calculation Example

```
Stock: ABC (3D Printing Sector)

W1 - Structure Score: 18/20
  Float 15M → 4/5
  Insider 35% → 5/5
  Institutional 8% → 5/5
  Price $6.50 → 4/5
  No Dilution → 0/0 (pass)

W2 - Technical Score: 27/30
  Basing 400 days → 9/10
  BB Width 8th percentile → 9/10
  RVOL 12x → 9/10

W3 - Sector Score: 18/20
  4 stocks breakout → 9/10
  Lead Dog identified → 9/10

W4 - Catalyst Score: 25/30
  Hard Catalyst (FDA) → 10/10
  Thematic (3D printing) → 8/10
  Not yet priced in → 7/10

TOTAL = (18×0.2) + (27×0.3) + (18×0.2) + (25×0.3)
      = 3.6 + 8.1 + 3.6 + 7.5
      = 22.8 / 30
      = 76%

ACTION: HIGH CONVICTION → Enter position
```

### Thresholds

| Score | Action | ความหมาย |
|-------|--------|----------|
| **> 75** | HIGH CONVICTION | Deep Dive ทันที พร้อมเข้า |
| **60-75** | WATCHLIST | รอ Technical setup |
| **< 60** | DISCARD | ไม่ตอบโจทย์ |

## Part 7 — Risk Management: กฎเหล็ก 5 ข้อ

### กฎเหล็กของ Neumann

**① Never Average Down**

```
หุ้นลงหลังซื้อ → ออกทันที

```

ผมเคย Average Down แล้วโดน Deep Down มาแล้ว อย่าทำ

```
หลีกเลี่ยงหุ้นที่คนพูดถึงเยอะ

```

ถ้าคนทั่วไปรู้แล้ว = รอช้าไปแล้ว

**③ No Hard Stops**

```
ใช้ mental stops ตาม thesis
ถ้า thesis break = Exit immediately
ไม่ใช่รอให้ราคาแตะ stop loss
```

Hard stops โดน Stop Hunt บ่อย ผมใช้ Mental stops ตาม Thesis แทน

**④ Know the Filings**

```
ตรวจ S-3, 424B, ATM keywords
```

**⑤ Wait for the Turn**

```
รอจนกว่า 4 Corners align

"ผมไม่ได้เล่นตลาด ผมรอให้ทุกอย่างเรียงตัวสวย"
```

## Part 10 — SEC Filings Deep Dive

Dilution = ศัตรูร้ายแรงที่สุด ผมต้องอ่านทุก Filing ก่อนจะซื้อ

### Filing ที่ต้องรู้

**① S-3 (Shelf Registration)**

```
S-3 = บริษัทยื่นขายหุ้นได้ "เมื่อไหร่ก็ได้"
ไม่ใช่ขายทันที แต่เป็นการเตรียมไว้ก่อน
```

**Keywords อันตรายใน S-3:**
```
- "We may offer and sell from time to time"
- "At-the-market offerings"
- "Sales agreement"
- "Equity distribution agreement"
```

**Action:** ถ้าเจอ S-3 ใหม่ → -30 pts, หรือ SKIP

**② 424B (Prospectus Supplement)**

```
424B = บริษัทกำลังขายหุ้นตอนนี้!
```

**Keywords อันตรายใน 424B:**
```
- "We are offering $X million shares"
- "This prospectus relates to"
- "Public offering price"
```

**Action:** ถ้าเจอ 424B ใหม่ → -50 pts, หรือ SKIP

**③ 8-K (Current Report)**

```
8-K = รายงานเหตุการณ์สำคัญ
ต้องดู Section ที่เกี่ยวกับหุ้น
```

**Keywords อันตรายใน 8-K:**
```
- "Entry into a Material Definitive Agreement"
- "Issuance of securities"
- "Private placement"
- "At-the-market offering agreement"
```

**Action:** ถ้าเจอ Keywords เหล่านี้ → Investigate further

### Dilution Red Flags Checklist

```
✅ CHECK BEFORE BUYING:
□ มี S-3 ล่าสุด (3 เดือน) ไหม?
□ มี 424B ล่าสุด (1 เดือน) ไหม?
□ 8-K มี Keywords อันตรายไหม?
□ Outstanding shares เพิ่มขึ้นเร็วๆ ไหม?
□บริษัทมีประวัติ Dilution บ่อยไหม?
```

## Part 11 — จิตวิทยาการรอ

### ทำไมคนส่วนใหญ่รีบเข้าแล้วพลาด

ผมดู Traders มาพันคน สิ่งที่แยกคนที่รอดจากคนที่โดน Cut มีอยู่อย่างเดียว: **ความอดทนในการรอ**

### FOMO = Fear Of Missing Out

```
คนทั่วไปเห็นหุ้นพุ่ง → FOMO → ซื้อเลย → Top
Neumann เห็นหุ้นพุ่ง → Check 4 Corners → รอ align → Bottom
```

### วิธีควบคุม FOMO

**① มี Checklist ชัดเจน**

```
ก่อนซื้อทุกครั้ง:
□ 4 Corners align หรือยัง?
□ Neumann Score > 75 หรือยัง?
□ มี Dilution ไหม?
ถ้าตอบ "ไม่" → รอ
```

**② ระบุ "อันตราย" ของการรอ**

```
รอ = พลาดจังหวะนี้ แต่
รีบ = โดน Cut = เสียเงินจริง
ผมเลือกพลาดจังหวะ ไม่เสียเงิน
```

**③ รู้ว่า "อีกจังหวะมีเยอะ"**

```
ตลาดมีหุ้น 10,000+ ตัว
จังหวะดีเกิดทุกวัน
รอสัก 2-3 วันไม่เสียหาย
แต่ FOMO ซื้อผิดจังหวะ = เสียเงินจริง
```

### กฎของผมเรื่องการรอ

```
1. 4 Corners ไม่ align = ไม่ซื้อ (ไม่ว่าราคาจะดีแค่ไหน)
2. Neumann Score < 75 = ไม่ซื้อ (รอให้ Score ขึ้น)
3. มี Dilution = ไม่ซื้อ (รอจนกว่า clear)
4. Sector ไม่ Cluster = ไม่ซื้อ (รอ 3+ stocks)
```

> 💬 **One-Liner:** "The market is a device for transferring money from the impatient to the patient" — ผมเลือกเป็นคน Patient

## Part 13 — คำถามที่พบบ่อย (FAQ)

**Q1: ต้องมีเงินเท่าไหร่เริ่มได้?**

A: ผมเริ่มด้วย $2,500 แต่ความจริงคุณสามารถเริ่มด้วย $1,000–$5,000 ก็ได้ สิ่งสำคัญคือ Risk Management และการรอจังหวะที่ดี

**Q2: ต้องดูกราฟทุกวันไหม?**

A: ไม่! ผมดู Daily Chart หลักๆ และ Review ทุกวัน แต่ไม่ Glue หน้าจอ 24/7

**Q3: ถ้า Sector ไม่ Breakout แต่หุ้นเดี่ยวดี ซื้อได้ไหม?**

A: ได้ แต่ลด Conviction ลง และลด Size ลง ผมชอบ Sector Breakout มากกว่าเพราะมี Momentum ร่วมกัน

**Q4: ถ้า RVOL ไม่พุ่ง แต่ทุกอย่างดี ซื้อได้ไหม?**

A: รอสิ! Volume ต้องพุ่งถึงจะ Breakout แท้ ไม่งั้น Fake Out

**Q5: ถ้า Thesis ยังดี แต่ราคาลง ทำไง?**

A: ถ้า Thesis ยังดี แต่ราคาลงเล็กน้อย = Hold แต่ถ้าราคาหลุดจุดสำคัญ (Support, Key level) = Thesis break = Exit

**Q6: ถ้าหุ้นขึ้นแรงมาก 50%+ ใน 1 สัปดาห์ ทำไง?**

A: ขายบางส่วน ล็อกกำไร และถือบางส่วนต่อ อย่าโลภ

**Q7: Dilution เป็นศัตรูร้ายเสมอไหม?**

A: เกือบตลอด แต่ถ้าบริษัทใช้เงินดี (ขยายธุรกิจ) และไม่ขายเยอะ อาจโอเค แต่ส่วนใหญ่ Dilution = ราคากด

**Q8: จะรู้ได้ไงว่า Sector อยู่ Phase ไหน?**

A: ดู 3 อย่าง:
1. **Volume** — Stealth = ต่ำ, Mania = พุ่ง
2. **Media Coverage** — Stealth = เงียบ, Mania = ทุกที่
3. **Public Sentiment** — Stealth = ไม่มีใครรู้, Mania = ทุกคนคุย

**Q9: Mental Stop vs Hard Stop ใช้ยังไง?**

A: ผมใช้ Mental Stop ตาม Thesis เช่น:
- Thesis: "FDA will approve" → If FDA delays = Exit immediately
- Thesis: "Sector momentum" → If Sector breaks down = Exit immediately

Hard stops โดน Stop Hunt บ่อย ผมชอบ Mental stops มากกว่า

**Q10: Pyramid Strategy ทำยังไง?**

A: ผมทำแบบนี้:
```
Initial: 10-15% position
If +20%: Add 10% more
If +50%: Add 10% more
If +100%: Add 5% more

Total: 40% max position
But only if Thesis still intact
```

## คำสั่งเสียจาก Jeffrey Neumann

1. **รอให้ทุกอย่างเรียงตัว** — 4 Corners ต้อง align ถึงจะเข้า
2. **อย่า Average Down** — หุ้นลงหลังซื้อ = Thesis break = ออก
3. **หลีกเลี่ยง Crowd** — ถ้าคนทั่วไปรู้แล้ว = รอช้าแล้ว
4. **รู้ Filings ให้แม่น** — Dilution = ศัตรู
5. **ซื้อ Sector ไม่ใช่หุ้น** — Shotgun Approach ช่วยกระจายความเสี่ยง
6. **Pyramid เข้า Lead Dog** — ตัวที่แรงสุดใน Sector
7. **Sell on News** — Buy Rumor, Sell News
8. **ใช้ Mental Stops** — ตาม Thesis ไม่ใช่ Hard Stop
9. **Patience > Action** — รอจังหวะที่ดีที่สุด
10. **Discipline > Intelligence** — คนฉลาดแต่ขาดวินัยโดน Cut
