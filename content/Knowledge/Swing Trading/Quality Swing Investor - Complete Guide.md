# Quality Swing Investor - ความหมายและแนวทางที่แท้จริง

> **การผสานความแข็งแกร่งระหว่าง Value + Growth + Momentum**
> **Source: Dual AI Debate (Codex + จานดัน)**
> **Date: 2026-03-29**

---

## 🎯 บทสรุปสั้น ๆ

**Quality Swing Investor คือนักลงทุนที่:**
> **"เลือกธุรกิจคุณภาพสูง (Quality) + จับจังหวะราคา (Swing) + มีวินัยเข้า-ออก (Investor)"**

**สูตร:**
```
พื้นฐานกำหนดว่า "ซื้ออะไร" + เทคนิคกำหนดว่า "ซื้อเมื่อไร-ขายเมื่อไร"
```

**คำกล่าวของจานดัน:**
> **"Quality Swing Investor ที่แท้จริง คือคนที่ 'ไม่ลดมาตรฐานธุรกิจ' เพื่อเอาจังหวะ และ 'ไม่ทิ้งวินัยจังหวะ' เพราะหลงรักธุรกิจ"**

---

## 📊 สาระสำคัญ 4 ประการ

| # | หลักการ | คำอธิบาย |
|---|---------|----------|
| 1 | **Quality First** | คัดเฉพาะกิจการที่โครงสร้างธุรกิจแข็งแรง มี Moat ชัดเจน |
| 2 | **Mispricing Window** | รอ "ช่วงผิดราคา" จากข่าวลบระยะสั้น/งบชะลอชั่วคราว |
| 3 | **Cycle Timing** | เข้าใกล้จุดฟื้นตัวของกำไร/โมเมนตัมราคา |
| 4 | **Rule-based Exit** | ขายเมื่อ thesis เล่นครบ, valuation ตึง, หรือสัญญาณเทคนิคเสีย |

---

## 🧠 ความรู้ที่จำเป็น

### 📈 Fundamental (ต้องแน่น)

| หัวข้อ | สิ่งที่ต้องรู้ |
|--------|--------------|
| **คุณภาพธุรกิจ** | Moat, Pricing Power, Switching Cost, Network Effect |
| **คุณภาพกำไร** | รายได้/กำไรโตจริงหรือโตจากรายการพิเศษ (ดู CFO, FCF, Accrual) |
| **งบดุล** | หนี้สิน, ดอกเบี้ยจ่าย, กระแสเงินสด, ความเสี่ยงเพิ่มทุน |
| **ผลตอบแทน** | ROIC, ROE, FCF Yield - บริษัทดีต้องคืนทุนเก่ง |
| **Valuation** | P/E, EV/EBITDA, PEG, DCF แบบง่าย, Historical Band |
| **Catalyst** | ปัจจัยเร่งราคาใน 1-4 ไตรมาส (margin ฟื้น, product cycle, utilization ขึ้น) |
| **Earnings Revision** | นักวิเคราะห์ปรับประมาณการขึ้น/ลง มีผลมากกับ swing |
| **Industry Cycle** | หุ้นดีแต่ซื้อผิดวัฏจักร ก็เจ็บได้ |

### 📉 Technical (ใช้เป็นเครื่องมือจังหวะ)

| หัวข้อ | สิ่งที่ต้องรู้ |
|--------|--------------|
| **Trend Structure** | Higher High/Higher Low, Trend Reversal |
| **Volume Analysis** | Breakout ต้องมี Volume ยืนยัน |
| **Momentum** | RSI/MACD ใช้ดูแรงส่ง ไม่ใช่สัญญาณเดี่ยว |
| **Key Levels** | แนวรับ-ต้าน, Base Breakout, Retest |
| **Risk Control** | จุดตัดขาดทุนจากโครงสร้างราคา ไม่ใช่ตั้งมั่ว |
| **Position Sizing** | ขนาดไม้ตาม Conviction + Volatility |
| **Market Context** | หุ้นดีแต่ตลาดเป็นขาลงแรง อัตราชนะจะลด |
| **Multi-timeframe** | Weekly ดูภาพใหญ่, Daily หา Entry |

## 💡 จิตวิญญาณ/แนวคิดที่ต้องมี

| # | แนวคิด | คำอธิบาย |
|---|--------|----------|
| 1 | **Owner Mindset + Trader Discipline** | คิดแบบเจ้าของกิจการ แต่ execute แบบนักเทรดมีวินัย |
| 2 | **Probabilistic Thinking** | ไม่มีคำว่า "ชัวร์" มีแต่ "คุ้มความเสี่ยงไหม" |

| 6 | **Process > Outcome** | ดีลขาดทุนได้ แต่กระบวนการต้องถูก |
| 7 | **Humility** | ตลาดถูกกว่า Ego เสมอ |
| 8 | **Journal ทุกดีล** | เพื่อหาจุดอ่อนจริง ไม่ใช่จำแบบเลือกเฉพาะสิ่งที่อยากจำ |

## 📝 Checklist ก่อนกดซื้อ

### ✅ Quality Check
- [ ] ROIC/ROE > 15% ต่อเนื่อง 3-5 ปี?
- [ ] FCF บวกสม่ำเสมอ?
- [ ] งบดุลแข็งแรง (Net Debt/EBITDA < 2x)?
- [ ] Moat ชัดเจน (Brand, Cost, Network, Switching)?
- [ ] Governance ดี (Audit Clean, No RPT Risk)?

### ✅ Valuation Check
- [ ] P/E < Historical Median?
- [ ] EV/EBITDA < Sector Average?
- [ ] PEG < 1.5 (Growth)?
- [ ] Margin of Safety > 20%?

### ✅ Catalyst Check
- [ ] มี Catalyst ชัดเจนใน 3-12 เดือน?
- [ ] Earnings Revision ขึ้น?
- [ ] Utilization/Margin ฟื้นตัว?
- [ ] Product Cycle ใหม่?

### ✅ Technical Check
- [ ] แนวโน้มขึ้น (Higher High/Higher Low)?
- [ ] Volume สนับสนุน Breakout?
- [ ] RSI ไม่ Overbought?
- [ ] ใกล้แนวรับสำคัญ?

### ✅ Risk Check
- [ ] กำหนด Stop Loss แล้ว?
- [ ] Position Size ไม่เกิน 5-10%?
- [ ] Max Loss/ดีล ไม่เกิน 1-2% พอร์ต?
- [ ] Thesis Invalidation Point ชัด?
