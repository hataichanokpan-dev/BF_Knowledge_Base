# สไตล์ Value Swing

> การเทรดระยะสั้นถึงกลางที่อิงจุดเข้าจาก value และจังหวะทางเทคนิค

## ความหมาย

Value Swing ผสาน:
- มูลค่าพื้นฐาน (P/BV < 1x, P/E ต่ำกว่า sector)
- จังหวะทางเทคนิค (support/resistance)
- การรับรู้ catalyst (ปันผล, เหตุการณ์)
- ระยะเวลาถือครองที่ชัดเจน (หลายสัปดาห์ถึงหลายเดือน)

## เกณฑ์

### ต้องมี
- [ ] P/BV < 1x OR P/E < ค่าเฉลี่ย sector
- [ ] กำหนดแนวรับชัดเจน
- [ ] มี catalyst ภายในช่วงถือครอง
- [ ] ระบุ stop loss ได้ชัด

### ยิ่งมีจะยิ่งดี
- [ ] Dividend yield > 4%
- [ ] ได้แรงหนุนจากดอกเบี้ย
- [ ] มีรูปแบบ mean reversion ในอดีต

## ตัวอย่างใน BF-Vault
- [[Land and Houses]] - P/BV 0.84x, support ที่ 3.55
- [[SC Asset]] - P/BV 0.31x, มี catalyst จากปันผล

## ทำไมแนวทางนี้ถึงใช้ได้
1. **Value Floor** - book value ช่วยรองรับ downside
2. **Technical Entry** - ซื้อที่แนวรับ ไม่ใช่ซื้อแบบสุ่ม
3. **Catalyst Timing** - ขายตาม event/price target

## ความเสี่ยง
- Value trap (book value ลดลง)
- แนวรับหลุด
- catalyst ไม่เป็นไปตามคาด
