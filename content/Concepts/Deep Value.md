# สไตล์หุ้น Deep Value

> การลงทุนในหุ้นที่ซื้อขายต่ำกว่ามูลค่าที่แท้จริงอย่างมีนัยสำคัญ

## ความหมาย

การลงทุนแบบ Deep Value เน้นหุ้นที่ซื้อขายด้วยส่วนลดรุนแรงเมื่อเทียบกับมูลค่าที่แท้จริง โดยมักวัดจาก:
- P/BV < 0.5x (ซื้อขายต่ำกว่าสินทรัพย์สุทธิ)
- P/E < 50% ของค่าเฉลี่ย sector
- มูลค่าตลาดต่ำกว่า net current asset value

## เกณฑ์

### ต้องมี
- [ ] P/BV < 0.5x
- [ ] book value เป็นบวก
- [ ] ไม่มีความเสี่ยงล้มละลายในระยะสั้น
- [ ] มี catalyst สำหรับ re-rating (ถ้ามีจะดีมาก)

### สัญญาณเตือน
- ติดกับดัก Value trap (ถูกถาวร)
- ธุรกิจกำลังถดถอย
- มีปัญหาทางบัญชี
- ไม่มีเส้นทางกลับไปทำกำไร

## ตัวอย่างใน BF-Vault
- [[SC Asset]] - P/BV 0.31x

## ทำไมแนวทางนี้ถึงใช้ได้
1. **Margin of Safety** - downside จำกัด
2. **Mean Reversion** - valuation มีแนวโน้มกลับสู่ภาวะปกติ
3. **Catalyst Potential** - ข่าวบวกเพียงเล็กน้อยก็อาจกระตุ้น re-rating ได้

## ความเสี่ยง
- Value trap
- ระยะเวลาถือครองยาว
- ธุรกิจเสื่อมถอย
- ต้นทุนค่าเสียโอกาส
