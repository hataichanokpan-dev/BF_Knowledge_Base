# Deep Value Thesis

> **"ซื้อดอลลาร์ในราคา 50 เซ็นต์ พร้อม catalyst เพื่อปลดล็อคส่วนลด"**

---

## 🎯 The Core Idea

Deep Value คือการซื้อสินทรัพย์ในราคา **ต่ำกว่า intrinsic value อย่างมีนัยสำคัญ** (50-70% discount) พร้อม **margin of safety** ที่ปกป้อง downside

---

## 📐 The Formula

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   BUY SIGNAL = Discount + Quality + Catalyst            │
│                                                         │
│   ┌─────────┐   ┌──────────┐   ┌──────────┐           │
│   │ Discount │ + │ Quality  │ + │ Catalyst │ = BUY    │
│   │  >30%    │   │ Checks   │   │ Exists   │           │
│   └─────────┘   └──────────┘   └──────────┘           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 🔑 3 Pillars of Deep Value

### 1️⃣ DISCOUNT (ราคาถูกแค่ไหน?)

| Metric | Deep Value Signal | Formula |
|--------|-------------------|---------|
| **P/BV** | < 0.5x | Price / Book Value |
| **NCAV** | Price < 0.67 × NCAV | (Current Assets - Liabilities) / Shares |
| **NNWC** | Price < NNWC | Cash + 0.75×AR + 0.5×Inv - Liabilities |

**ความหมาย:** ซื้อสินทรัพย์ในราคาต่ำกว่ามูลค่าของสินทรัพย์นั้นจริงๆ

### 2️⃣ QUALITY (ไม่ใช่หุ้นแย่ที่ถูก)

**Gate 0 - Knockout Criteria (ต้องผ่านทั้งหมด):**

| Check | Why It Matters |
|-------|----------------|
| SET ESG Rating exists | Baseline governance |
| Free Float ≥ 20% | Liquidity + fair pricing |
| No trading warnings | No regulatory issues |
| Clean audit opinion | Financial integrity |

**Thai Market Critical Check:**

```
RPT (Related Party Transactions) Risk:
┌──────────┬──────────────┬─────────────────────┐
│  GREEN   │  < 10%       │ ✅ Proceed           │
│  YELLOW  │  10-20%      │ ⚠️ Investigate more │
│  RED     │  > 20%       │ ❌ AVOID             │
└──────────┴──────────────┴─────────────────────┘
```

### 3️⃣ CATALYST (อะไรจะปลดล็อคมูลค่า?)

| Catalyst Type | Example |
|---------------|---------|
| **Asset Sale** | ขายที่ดิน, monetize land bank |
| **Restructuring** | เปลี่ยน CEO, ลด cost |
| **Sector Rotation** | ตลาดหมุนกลับมาสนใจ sector |
| **M&A** | ถูก acquire, take private |
| **Buyback** | บริษัทซื้อหุ้นตัวเอง |

## 📊 The 4-Phase Funnel

```
PHASE 0: GATE 0 (Knockout)
    │
    ├── FAIL → ❌ STOP (no further analysis)
    │
    └── PASS
         │
         ▼
PHASE 1: QUANTITATIVE (Metrics)
    │
    ├── P/BV < 0.5x OR NCAV < 0.67×Price?
    │
    ├── NO → ❌ TOO EXPENSIVE
    │
    └── YES
         │
         ▼
PHASE 2: THAI CHECKS (Governance)
    │
    ├── RPT RED (>20%)?
    │
    ├── YES → ❌ GOVERNANCE RISK
    │
    └── NO
         │
         ▼
PHASE 3: ENTRY (Price)
    │
    ├── Current Price ≤ 0.70 × Intrinsic Value?
    │
    ├── NO → 🟡 WATCH (wait for better price)
    │
    └── YES → ✅ BUY
```

## ⚠️ Why Deep Value Works (และทำไมมัน Risky)

### Why It Works

| Reason | Explanation |
|--------|-------------|
| **Margin of Safety** | ซื้อที่ 50-70% ของ IV = buffer ต่อการวิเคราะห์ผิด |
| **Mean Reversion** | P/BV 0.3x มักกลับสู่ 0.8-1.2x เอง |
| **Asymmetric Risk** | Downside จำกัด (asset floor), upside สูง |
| **Catalyst Leverage** | ข่าวดีเล็กน้อย → re-rating ได้มาก |

### Why It's Risky

| Risk | Mitigation |
|------|------------|
| **Value Trap** | ต้องมี catalyst + business ยัง viable |
| **Governance** | RPT check, board independence |
| **Time Cost** | อาจถือ 2-3 ปีจน catalyst เกิด |
| **Liquidity** | Free float check, position size control |

## 📜 Academic Evidence (งานวิจัยสนับสนุน)

### หลักฐานเชิงวิชาการ (Strongest Evidence)

#### 1. Fama & French (1992, 1993) - The Value Factor

**Study:** "The Cross-Section of Expected Stock Returns"

**Key Finding:**
- ระบุปัจจัย **HML (High Minus Low)** ใน 3-factor model
- หุ้นที่มี Book-to-Market สูง (Value) ให้ผลตอบแทนสูงกว่าหุ้น B/M ต่ำ (Growth)
- Value premium อธิบายว่าเป็นค่าตอบแทนจากความเสี่ยง (distress risk) ที่สูงขึ้น

**Implication:** Value factor เป็น systematic risk factor ที่มีอยู่จริงและวัดได้

---

#### 2. Lakonishok, Shleifer, Vishny (1994) - Behavioral Explanation

**Study:** "Contrarian Investment, Extrapolation, and Risk"

**Key Finding:**
- Value premium มาจาก **การตั้งราคาผิดของนักลงทุน** (mispricing)
- นักลงทุนมีแนวโน้ม **ประเมินค่าเกินจริง** การเติบโตในอดีตของหุ้น Growth
- และ **ประเมินค่าต่ำเกินไป** หุ้น Value ที่มีผลประกอบการไม่ดีในระยะสั้น

**Implication:** Value investing ไม่ใช่รับความเสี่ยง แต่รับจากการตัดสินใจที่ผิดพลาดของตลาด

---

#### 3. Oppenheimer (1986) - Graham's NCAV Performance

**Study:** "Ben Graham's NCAV: Performance Study 1970-1981"

**Key Finding:**
- Graham's Net-Net strategy (Price < NCAV) ให้ผลตอบแทน **20-30% CAGR**
- ทดสอบในหลายทศวรรษ ยังคงให้ผลตอบแทนเหนือกว่าตลาด
- หุ้นกลุ่มนี้มักมีขนาดเล็กและสภาพคล่องต่ำ

**Implication:** NCAV strategy ยังใช้ได้จริง แม้ในยุคสมัยใหม่

---

#### 4. Piotroski (2000) - Quality Filter Enhancement

**Study:** "Value Investing: The Use of Historical Financial Statement Information"

**Key Finding:**
- การใช้ **F-Score** (9 ตัวชี้วัดทางการเงิน) ในการคัดกรองหุ้น Value
- หุ้น Value ที่มี F-Score สูง **เพิ่มผลตอบแทนได้ถึง 7.5% ต่อปี**
- แสดงว่าการวิเคราะห์พื้นฐาน (fundamental analysis) เพิ่มประสิทธิภาพ Deep Value ได้อย่างมีนัยสำคัญ

**Implication:** Deep Value + Quality Filter = Alpha ที่สูงขึ้น

---

#### 5. Tweedy Browne - Compiled Evidence

**Paper:** "What Has Worked in Investing"

**Key Finding:**
- รวบรวมงานวิจัยหลายชิ้นที่แสดง NCAV strategies ให้ **15-20%+ annual returns**
- ระบุปัจจัยสำคัญ: small caps, low liquidity, longer holding periods

**Implication:** Deep Value เป็นกลยุทธ์ที่มีหลักฐานยืนยันจากหลาย sources

### ข้อวิพากษ์และข้อจำกัด (Criticisms)

#### 1. Value Trap

**ปัญหา:** หุ้นถูกเพราะปัจจัยพื้นฐานแย่ ไม่ใช่ mispricing

**รายละเอียด:**
- ธุรกิจล้มเหลว (business failure)
- หนี้สินสูง (excessive debt)
- การบริหารจัดการที่ไม่ดี (poor management)
- อุตสาหกรรมอยู่ในช่วง structural decline

**นักวิชาการบางท่านเชื่อว่า:** "หุ้นที่ถูกที่สุดมักจะถูกด้วยเหตุผล"

**Mitigation:** ต้องมี catalyst + business ยัง viable + quality checks

---

#### 2. Convergence Risk (Duration Risk)

**ปัญหา:** ไม่รู้ว่าราคาจะกลับสู่ IV เมื่อไหร่ และจะเกิดขึ้นจริงหรือไม่

**รายละเอียด:**
- อาจรอ 2-3 ปีจน catalyst เกิด
- ต้นทุนการถือคือ opportunity cost
- บางครั้งราคาไม่มาบรรจบเลย (permanently impaired)

**Mitigation:** ต้องมี catalyst ที่ระบุชัดเจน + time horizon ที่ยอมรับ

---

#### 3. Intangible Assets Problem

**ปัญหา:** P/BV ไม่ capture มูลค่าของสินทรัพย์ไม่มีตัวตน

**รายละเอียด:**
- บริษัทที่มี R&D, แบรนด์, software มาก
- P/BV จะดูแพงเกินจริงเพราะ Book Value ต่ำ
- วิธีแก้: ใช้ EV/EBITDA หรือ P/FCF แทน

**Mitigation:** ปรับ metric ตามธุรกิจ ใช้ sum-of-parts สำหรับ asset-heavy

---

#### 4. Information Risk (Emerging Markets)

**ปัญหา:** ในตลาดเกิดใหม่ คุณภาพข้อมูลอาจไม่น่าเชื่อถือ

**รายละเอียด:**
- การเปิดเผยข้อมูลทางการเงินไม่ดี
- การตรวจสอบบัญชีอ่อนแอ
- Related Party Transactions ซ่อนเร้น
- Minority shareholders อาจถูกเอาเปรียบ

**Mitigation:** Gate 0 + governance checks ที่เข้มงวดขึ้น

---

### พัฒนาการล่าสุด 2020-2025

#### The "Value Is Dead" Debate (2020)

**ช่วงต้นปี 2020:**
- Value Premium ประสบภาวะตกต่ำอย่างรุนแรง vs Growth
- Tech stocks ขึ้นสูงมาก
- คำถาม: "Value ตายแล้วหรือไม่?"

**ตอบ:** ยังไม่ตาย แต่เป็น **regime-dependent**

---

#### Value Resurgence (2021-2022)

**ช่วง 2021-2022:**
- อัตราเงินเฟ้อและดอกเบี้ยสูงขึ้น
- Sector rotation กลับมายังหุ้น Value
- Energy, Financials, Industrials ทำผลตอบแทนดี
- Value กลับมา outperform อีกครั้ง

---

#### AI Growth Wave (2023-2024)

**ช่วง 2023-2024:**
- AI stocks (NVDA, MSFT, GOOGL) นำตลาด
- Growth กลับมา outperform
- แต่นี่คือ **cycle ไม่ใช่ structural change**
- Value premium ยังคงอยู่แต่ต้องรอ timing

---

#### Regime Shifts Research (2025)

**งานวิจัยล่าสุด:**

| Paper | Finding |
|-------|---------|
| **Monge et al. (2025)** | ตลาดเปลี่ยนจาก "high persistence" ของ Growth เป็น "moderate persistence" ที่มีการกลับสู่ค่าเฉลี่ย (mean reversion) ของ Value |
| **Li et al. (2025)** | **Implied Value Premium (IVP)** ซึ่งอ้างอิงจากต้นทุนเงินทุน เป็นตัวบ่งชี้ที่ดีถึงผลตอบแทน Value ในอนาคต ส่วนต่างราคาที่กว้างในปัจจุบันยังบ่งบอกถึงโอกาสของ Value |

**Implication:** Value premium ไม่ได้หายไป แต่เป็นปัจจัยที่ขึ้นอยู่กับสภาวะตลาด (regime-dependent) อย่างมาก

### Thai Market Specific

> ⚠️ **งานวิจัยพบว่า Value Premium ในตลาดไทยมีลักษณะ:**

#### 1. ไม่ชัดเจนเท่าตลาดพัฒนาแล้ว

- Growth stocks มักให้ผลตอบแทนดีกว่า Value ในไทย
- ตลาดไทยมีความผันผวนสูง (high turnover)
- Retail investors มีสัดส่วนสูง

#### 2. Cyclical สูง

- Value premium ขึ้นอยู่กับช่วงเวลา (timing matters)
- ช่วงที่ดี: Post-crisis, Rate cycle upturn
- ช่วงที่ไม่ดี: Bull market, Growth rally

#### 3. ต้องใช้ Quality Filter

- **F-Score + governance checks จำเป็นมาก**
- ต้องตรวจ RPT, audit quality, board independence เข้มงวดขึ้น
- Gate 0 methodology สร้างขึ้นสำหรับตลาดไทยโดยเฉพาะ

| ไต้หวัน | ⚠️ ไม่ชัดเจน | Tech-heavy market |

## 🛡️ Risk Management

### Value Trap Detection Protocol

> **Deep Value มีความเสี่ยงสูงกว่า Dividend Play** เพราะไม่มี cash flow หนุนระหว่างรอ

#### 🚫 Red Flags (Knockout Criteria)

| Red Flag | Score | Trigger | Why It Matters |
|----------|-------|---------|----------------|
| **Business Failure** | 10 | กำไรขาดทุน 3 ปีติด | ธุรกิจอาจล้มเหลว |
| **No Catalyst** | 9 | ไม่มี catalyst ที่ระบุได้ใน 2 ปี | ราคาอาจไม่มาบรรจบ IV |
| **High Debt** | 8 | D/E > 2.0x หรือ Interest Coverage < 1.5x | ขาดสภาพคล่อง, default risk |
| **Governance Red** | 9 | RPT > 20% หรือ Audit ไม่ผ่าน | Minority shareholders ถูกเอาเปรียบ |
| **Sector Decline** | 8 | อุตสาหกรรมอยู่ใน structural decline | Asset value อาจลดลงเรื่อยๆ |
| **Negative FCF** | 7 | FCF ติดลบ 3 ปีติด | ต้องขาย assets หรือ raise capital |

#### ⚠️ Yellow Flags (Caution Criteria)

| Yellow Flag | Score | Trigger | Action |
|-------------|-------|---------|--------|
| **Low Liquidity** | 4 | Free Float < 20% หรือ Volume < 1M/day | ยากขาย, position ต้องเล็ก |
| **Convergence Risk** | 4 | Catalyst ไม่ชัดเจน | ต้องถือนาน, opportunity cost |
| **Asset Quality** | 3 | NCAV มี inventory/receivables สูง | อาจ write-down |
| **Management Track Record** | 3 | ไม่มีประวัติสร้าง value | Catalyst อาจไม่เกิด |

#### 📊 DV-VTS Calculator (Deep Value Value Trap Score)

```
DV-VTS = Σ(Red Flag Scores) + Σ(Yellow Flag Scores)

Decision Rule:
DV-VTS ≥ 15  →  🚫 REJECT (high probability value trap)
DV-VTS 8-14  →  ⚠️ CAUTION (deep dive required)
DV-VTS < 8   →  ✅ PASS (proceed to position sizing)
```

#### 🔍 Value Trap vs Genuine Deep Value

| Characteristic | Value Trap | Genuine Deep Value |
|----------------|------------|-------------------|
| **Earnings** | ขาดทุนต่อเนื่อง | กำไรลดแต่ยังบวก หรือกำลังฟื้น |
| **Debt** | สูงและเพิ่มขึ้น | คงที่หรือลดลง |
| **Asset Quality** | Inventory/AR สูง, ต้อง write-down | Cash + Real Estate สูง |
| **Management** | ไม่มีแผน, สับสน | มี catalyst plan ชัดเจน |
| **Sector** | Declining | Cyclical downturn หรือ temporarily out of favor |

### Position Sizing Framework

> **Deep Value ต้องใช้ position เล็กกว่า Dividend Play** เพราะ illiquidity และ uncertainty สูงกว่า

#### 📐 Base Position Tiers

| Tier | Base | Criteria | Description |
|------|------|----------|-------------|
| **A** | 5% | P/BV < 0.3x + Catalyst confirmed + High liquidity | High conviction, clear catalyst |
| **B** | 3% | P/BV 0.3-0.5x + Catalyst likely + Medium liquidity | Standard deep value |
| **C** | 2% | P/BV 0.5-0.7x + Catalyst uncertain OR Low liquidity | Speculative, small position |

#### 🔧 Adjustments

| Factor | Adjustment | Why |
|--------|------------|-----|
| **Illiquidity** | -50% if Free Float < 20% | ยากขาย |
| **No Catalyst** | -50% if no clear catalyst | Convergence risk |
| **High MOS (>50%)** | +25% if Price < 0.5 × IV | Extra margin of safety |
| **F-Score > 7** | +25% if Piotroski F-Score > 7 | Quality confirmation |

#### 📐 Final Position Formula

```
Final Position = Base Tier × Adjustments × MOS Factor

Example:
- Tier B (3%)
- Free Float 15% (-50%)
- F-Score 8 (+25%)
- MOS 60% (+25%)

Final = 3% × 0.5 × 1.25 × 1.25 = 2.34%
```

#### ⚠️ Hard Limits

| Rule | Limit | Why |
|------|-------|-----|
| **Max per Position** | 5% | Single stock risk |
| **Max Deep Value Allocation** | 20% of Portfolio | Overall strategy risk |
| **Min Cash Reserve** | 10% | Opportunity fund for better prices |
| **Max Low Liquidity Positions** | 3 stocks | Liquidity risk management |
