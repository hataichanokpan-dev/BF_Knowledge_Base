## 🔬 Core Finding

### F-Score Adds 7.5% Annually

> **การใช้ F-Score ในการคัดกรองหุ้น Value เพิ่มผลตอบแทนได้ถึง 7.5% ต่อปี**

Piotroski แสดงว่า:
1. หุ้น Value ทั้งหมดให้ผลตอบแทนสูงกว่าตลาด
2. แต่หุ้น Value ที่มี F-Score สูงให้ผลตอบแทนดีกว่ามาก
3. Fundamental analysis เพิ่มประสิทธิภาพ Deep Value ได้อย่างมีนัยสำคัญ

## 📐 The F-Score System

### 9 Criteria (1 Point Each)

```
F-Score = Sum of 9 Binary Signals (0-9 points)

┌─────────────────────────────────────────────────────────────┐
│                    PROFITABILITY (4 points)                  │
├─────────────────────────────────────────────────────────────┤
│ 1. ROA > 0                     → +1 (profitable)           │
│ 2. OCF > 0                     → +1 (cash positive)        │
│ 3. ΔROA > 0                    → +1 (improving)            │
│ 4. OCF > Net Income            → +1 (quality earnings)     │
├─────────────────────────────────────────────────────────────┤
│                    LEVERAGE (3 points)                      │
├─────────────────────────────────────────────────────────────┤
│ 5. ΔLong-term Debt < 0         → +1 (deleveraging)         │
│ 6. ΔCurrent Ratio > 0          → +1 (liquidity improving)  │
│ 7. ΔShares Outstanding ≤ 0     → +1 (no dilution)          │
├─────────────────────────────────────────────────────────────┤
│                    OPERATING EFFICIENCY (2 points)          │
├─────────────────────────────────────────────────────────────┤
│ 8. ΔGross Margin > 0           → +1 (margin improving)     │
│ 9. ΔAsset Turnover > 0         → +1 (efficiency improving) │
└─────────────────────────────────────────────────────────────┘
```

## 📊 The Evidence

### Performance by F-Score (High B/M Stocks)

| F-Score | Annual Return | Excess vs Low F-Score |
|---------|---------------|----------------------|
| **0-3 (Low)** | 5.7% | — |
| **4-6 (Mid)** | 11.4% | +5.7% |
| **7-9 (High)** | 17.3% | +11.6% |

**Key Finding:** High F-Score Value = +7.5% vs Average Value

## 💡 Key Insights

### 1. Why F-Score Works

| Signal | What It Catches |
|--------|-----------------|
| **Profitability** | Business still viable |
| **Cash Quality** | Not accounting gimmicks |
| **Deleveraging** | Financial distress reducing |
| **No Dilution** | Shareholder-friendly |
| **Efficiency** | Turnaround potential |

---

## 📐 Thai Market Application

### Example Calculation

```
Thai Company F-Score Example:

1. ROA = 3.2% > 0         ✅ +1
2. OCF = 450M > 0         ✅ +1
3. ΔROA = +1.5% > 0       ✅ +1
4. OCF (450) > NI (180)   ✅ +1
5. ΔLT Debt = -50M < 0    ✅ +1
6. ΔCurrent Ratio = +0.2  ✅ +1
7. ΔShares = 0 (no issue) ✅ +1
8. ΔGross Margin = +2.1%  ✅ +1
9. ΔAsset Turnover = +0.05✅ +1

F-Score = 9/9 → STRONG BUY CANDIDATE
```

## ⚠️ Caveats

### 1. Accounting Quality

> F-Score ใช้ accounting data
>
> ในตลาดไทย ต้องเช็ค audit quality ด้วย

---

### 2. Industry Differences

| Industry | F-Score Bias |
|----------|--------------|
| **Tech** | Low asset turnover (intangibles) |
| **Retail** | High turnover, low margin |
| **Property** | Low turnover, high margin |
| **Banking** | Different metrics needed |

---

### 3. Not Standalone

> F-Score ควรใช้ร่วมกับ:
> - Valuation (P/BV, NCAV)
> - Governance (RPT, Board)
> - Catalyst (Asset sale, Restructuring)
