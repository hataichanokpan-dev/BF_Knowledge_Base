## 🚨 Gate 0: Knockout Rules (ตัดทันที)

ถ้ามีข้อใดข้อหนึ่ง → **PASS ทันที ไม่ต้องวิเคราะห์ต่อ**

| # | Rule | หมายเหตุ |
|---|------|---------|
| 1 | มี sign `SP / NC / CB / CS / CF / CC` | จาก SET |
| 2 | Auditor opinion: Qualified / Adverse / Disclaimer | งบมีปัญหา |
| 3 | CFO ติดลบ 2 ปีติดต่อกัน | กำไรคุณภาพต่ำ |
| 4 | มูลค่าซื้อขายเฉลี่ย 20 วัน < 10 ลบ./วัน | สภาพคล่องต่ำ |
| 5 | ผู้บริหารมี red flag (ถูกดำเนินคดี, ขายหุ้นบิ๊กล็อต) | Governance |
| 6 | มีการเพิ่มทุน (PP/RO) ที่จะทำให้ dilution > 10% ใน 1 ปี | กินผลประโยชน์ |

## 🚦 Gate 1: Quality Pass/Fail (50% Weight)

**ต้องได้อย่างน้อย 30/50 คะแนนจึงจะผ่าน**

### 1.1 Moat (10 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 9-10 | ผู้นำ Top 1-3 ในอุตสาหกรรม, recurring revenue >50%, margin ทรง/ดีขึ้น 3 ปี |
| 7-8 | ตำแหน่งตลาดแข็งแรง, margin ทรงตัว |
| 5-6 | สินค้าแทนกันง่าย, margin เริ่มหด |
| 0-4 | ส่วนแบ่งตลาดลด, margin หดแรง >5ppt ใน 2 ปี |

### 1.2 Financial Health (10 คะแนน)

| คะแนน | เกณฑ์ (SET ทั่วไป) | Bank | Property |
|-------|-------------------|------|----------|
| 9-10 | Net D/E <0.8x, ICR >6x, ROE >15%, CFO บวก 3/3 ปี | CAR >17%, NPL <3%, Coverage >150% | Net IBD/E <1.2x, ICR >3x |
| 7-8 | Net D/E 0.8-1.2x, ICR 4-6x, ROE 10-15% | CAR 14-17%, NPL 3-3.5% | Net IBD/E 1.2-1.5x |
| 5-6 | Net D/E 1.2-2.0x, ICR 2-4x | CAR <14%, NPL >3.5% | Net IBD/E >1.5x |
| 0-4 | Net D/E >2x, ICR <2x, CFO ติดลบ | ขาดทุน, NPL เร่งขึ้น | ผิดนัดชำระหนี้ |

### 1.3 Growth (10 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 9-10 | Revenue CAGR >10%, EPS CAGR >12%, 4Q YoY ยังบวก |
| 7-8 | Revenue/EPS CAGR 5-10% |
| 5-6 | โตต่ำกว่า 5% หรือผันผวน |
| 0-4 | รายได้หรือกำไรลด 2 ปีติด |

### 1.4 Management (10 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 9-10 | Clean audit, dilution <3%/ปี, execution ตามแผน, ไม่มี RPT แปลกๆ, มี buyback/insider buy |
| 7-8 | Clean audit, dilution <=5% |
| 5-6 | Miss guidance, dilution 5-10%, management turnover |
| 0-4 | Qualified audit, dilution >10%, governance risk ชัด |

### 1.5 Valuation (10 คะแนน)

| คะแนน | เกณฑ์ (SET ทั่วไป) | Bank | Property | Tech |
|-------|-------------------|------|----------|------|
| 9-10 | P/E <=14.5x หรือ P/BV <=1.1x พร้อม ROE >=12%, PEG <1.0 | P/BV 0.7-1.0x + ROE >=9% | P/E 5-8x หรือ P/BV 0.5-0.9x | PEG <1.5, EV/EBITDA <15x |
| 7-8 | P/E 14.5-18x หรือ P/BV 1.1-1.5x | P/BV 1.0-1.2x | P/E 8-10x หรือ P/BV 0.9-1.1x | PEG 1.5-2.0 |
| 5-6 | P/E 18-24x หรือ P/BV 1.5-2.0x | P/BV 1.2-1.5x | P/E >10x หรือ P/BV >1.2x | EV/EBITDA >20x |
| 0-4 | P/E >24x หรือ P/BV >2x โดย growth ไม่รองรับ | P/BV >1.5x + ROE <7% | P/BV >1.5x แต่ presales ลด | P/E >40x แต่ growth <15% |

## 🎯 Gate 2: Catalyst Score (30% Weight)

### 2.1 Events (7.5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 7-7.5 | Catalyst ยืนยันแล้วใน 1-2 ไตรมาส และมีโอกาสดัน EPS/Revenue >10% |
| 5-6 | Catalyst ชัดใน 3-4 ไตรมาส |
| 3-4 | มี story แต่ยังไม่ confirm |
| 0-2 | ไม่มี catalyst หรือมี negative catalyst |

**Catalyst Examples:**
- [ ] งบเบิกเกินคาด (Earnings Surprise)
- [ ] ปันผลสูง / XD date ใกล้
- [ ] M&A / JV / ขยายกำลังผลิต
- [ ] นโยบายรัฐ (Digital Wallet, EEC, ลดดอกเบี้ย)
- [ ] ดอกเบี้ยขาลง (เช่าซื้อ, Property)

### 2.2 Industry (7.5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 7-7.5 | Sector index ชนะ SET >5ppt ใน 3M หรือ 6M, macro หนุนชัด |
| 5-6 | Sector ใกล้เคียง/ดีกว่า SET เล็กน้อย |
| 3-4 | Sector อ่อนกว่าตลาดเล็กน้อย |
| 0-2 | Sector แพ้ SET >5ppt และมี headwind |

### 2.3 Sentiment (7.5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 7-7.5 | EPS revision ขึ้น >5% ใน 90 วัน, NVDR/Foreign net buy 4 ใน 6 สัปดาห์ |
| 5-6 | Revision ทรงตัวถึงบวกเล็กน้อย |
| 3-4 | มุมมองคละกัน |
| 0-2 | Revision ลง >5%, มี overhang ชัด |

### 2.4 Risk Profile (7.5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 7-7.5 | ไม่มี sign เตือน, audit clean, concentration <25%, ไม่มี regulatory risk |
| 5-6 | ความเสี่ยงปกติของธุรกิจ |
| 3-4 | Concentration 25-35% หรือ covenant เริ่มตึง |
| 0-2 | Concentration >35%, regulatory risk, มี SP/NC risk |

## 📈 Gate 3: Swing Score (20% Weight)

### 3.1 Trend (5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 5 | ราคา > EMA20 > EMA50 > EMA200 และอยู่เหนือ 200DMA >10% |
| 4 | ราคาเหนือ EMA20/50/200 แต่ spacing ยังไม่สวยสุด |
| 2-3 | เหนือ 200DMA แต่ short-term ยังแกว่ง |
| 0-1 | ต่ำกว่า 200DMA |

### 3.2 Momentum (5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 5 | RSI 55-70 และ Relative Strength vs SET >+10ppt ใน 3M |
| 4 | RSI 50-60, RS ดีกว่า SET เล็กน้อย |
| 2-3 | RSI กลางๆ 45-55 |
| 0-1 | RSI <45 และ RS แพ้ SET ชัด |

### 3.3 Support/Resistance (5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 5 | Breakout เหนือ high 6 เดือน พร้อม close >2% และ retest ไม่หลุด |
| 4 | ใกล้ support ในระยะ 1-3% และ R/R >3:1 |
| 2-3 | อยู่กลางกรอบ |
| 0-1 | หลุด support หรือมี overhead supply ใกล้มาก |

### 3.4 Volume (5 คะแนน)

| คะแนน | เกณฑ์ |
|-------|-------|
| 5 | Breakout volume >=2x avg 20D, มูลค่าซื้อขาย avg 20D >=50 ลบ./วัน |
| 4 | Volume 1.5-2x, value 20-50 ลบ./วัน |
| 2-3 | Value 10-20 ลบ./วัน |
| 0-1 | Value <10 ลบ./วัน หรือมี gap จากสภาพคล่องต่ำ |

## ✅ Green Flags (บวกคะแนน)

- [ ] กำไรโต + margin ขยายพร้อมกัน (ไม่ใช่โตปลอม)
- [ ] ราคา breakout แล้วพักตัวไม่หลุด (demand จริง)
- [ ] Buyback / insider buy
- [ ] ไม่มี sign เตือนจาก SET
- [ ] Earnings revision ขึ้น 2 รอบติด
- [ ] หุ้นชนะ SET และชนะ sector พร้อมกัน
- [ ] Property: backlog สูง + โอนชัด
- [ ] Bank: NPL ต่ำกว่าระบบ + coverage สูง
- [ ] Tech: growth สูงแต่ valuation ยังไม่สุดโต่ง

## 🚩 Red Flags (หักคะแนนหรือ PASS)

- [ ] SP/NC/CB/CS/CF/CC → **PASS**
- [ ] Qualified/Adverse/Disclaimer audit → **PASS**
- [ ] Dilution หนัก, PP/RO ถี่
- [ ] มูลค่าซื้อขาย <10 ลบ./วัน → ห้าม BUY
- [ ] กำไรโตแต่ CFO ไม่มา 2 ปีติด
- [ ] Margin ลด 2 ไตรมาสติด + inventory พุ่ง
- [ ] หนี้สูง + ดอกเบี้ยกดกำไร
- [ ] ลูกค้ารายใหญ่ >35% ของรายได้
- [ ] Bank: NPL เร่งขึ้น / coverage ลด
- [ ] Property: presales อ่อน + backlog บาง
- [ ] Tech: valuation แพงแต่ growth ชะลอ
